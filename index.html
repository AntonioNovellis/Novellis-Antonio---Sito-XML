COS'E' L'XML ?

XML (eXtensible Markup Language) è un linguaggio di markup utilizzato per descrivere, strutturare e trasportare dati. A differenza di HTML, XML non serve a visualizzare contenuti, ma a organizzare informazioni in modo chiaro e leggibile sia per le persone che per i computer. È molto usato nello scambio di dati tra applicazioni e sistemi diversi.

LE REGOLE PRINCIPALI

Nel linguaggio XML ci sono delle regole fondamentali. Queste sono le principali.

  • 01.

    Nell'XML c'e' solo un elemento radice

  • 02.

    Ogni tag che si apre si deve sempre chiudere

  • 03.

    I tag XML sono sensibili alle maiuscole

  • 04.

    Gli elementi devono essere annidati

  • 05.

    Gli attributi devono avere valori tra virgolette

PERCHE' USARE L'XML ?

XML viene utilizzato perché è leggibile sia dall’uomo che dalle macchine, è indipendente dalla piattaforma e permette di scambiare dati tra sistemi diversi.

  • Leggibile da persone e computer
  • Indipendente dal sistema operativo
  • Ideale per lo scambio di dati
  • Estendibile con tag personalizzati
  • Usato in web services, API e database

XML VS HTML

Anche se XML e HTML sembrano simili, hanno scopi completamente diversi.

  1. XML

  2. Descrive i dati

  4. Struttura rigorosa
  5. Dati organizzati
